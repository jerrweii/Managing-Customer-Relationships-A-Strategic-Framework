# Skill 06: Customer Value Differentiation

## Purpose

Segment and prioritize customers by current value, potential value, referral value, and cost-to-serve.

## When To Use

- Designing scoring models.
- Prioritizing service, sales, retention, or success workflows.
- Creating customer tiers.

## Inputs

- Revenue and margin data.
- Product usage.
- Support cost.
- Retention history.
- Referral or influence signals.

## Repeatable Steps

1. Define value dimensions: current, lifetime, potential, referral, and cost.
2. Choose scoring windows and update frequency.
3. Normalize value scores for comparison.
4. Link value tiers to treatment policies.
5. Monitor whether treatment changes customer value over time.

## Software Artifacts

- Customer value score.
- Value tier model.
- LTV input pipeline.
- Prioritization rules.
- Value movement dashboard.

## Acceptance Checks

- Value scores are explainable.
- Tiers drive specific CRM actions.
- High-value and high-potential customers are distinguishable.
- The model accounts for cost-to-serve where relevant.
