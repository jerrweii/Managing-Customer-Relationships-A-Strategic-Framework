# Skill 16: Customer Lifetime Value Measurement

## Purpose

Measure customer value over time so CRM decisions optimize long-term relationship value.

## When To Use

- Prioritizing sales, success, retention, or service investment.
- Building value-based segmentation.
- Measuring CRM impact.

## Inputs

- Revenue.
- Margin.
- Retention.
- Cost-to-serve.
- Expansion, referral, or potential value data.

## Repeatable Steps

1. Define the LTV formula and assumptions.
2. Choose historical and forecast windows.
3. Identify value drivers and leading indicators.
4. Update LTV consistently and explain changes.
5. Tie LTV movement to CRM actions where possible.

## Software Artifacts

- LTV model.
- Value driver table.
- Customer value history.
- LTV dashboard.
- Attribution analysis.

## Acceptance Checks

- LTV is not confused with short-term revenue.
- Inputs and assumptions are visible.
- Value changes can be traced over time.
- LTV informs treatment and investment decisions.
