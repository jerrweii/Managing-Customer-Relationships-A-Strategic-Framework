# CRM Reusable Skills

This folder distills concepts from `Managing Customer Experience and Relationships: A Strategic Framework` into repeatable software development skills for CRM system design.

Use these files as a design checklist when planning, building, reviewing, or refactoring CRM capabilities.

## Core Loop

```text
Customer Strategy
-> Identify
-> Differentiate by Value and Need
-> Interact
-> Customize
-> Preserve Trust
-> Measure Customer Value
-> Govern the Enterprise
```

## How To Use

1. Start with `00-crm-system-design-skill-index.md`.
2. Pick the skill that matches the CRM feature under design.
3. Use the skill's inputs, steps, software artifacts, and acceptance checks.
4. During review, ask: does this feature improve identification, differentiation, interaction, customization, trust, value measurement, or governance?
