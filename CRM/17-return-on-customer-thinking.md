# Skill 17: Return on Customer Thinking

## Purpose

Track whether the company creates or consumes customer value over time.

## When To Use

- Evaluating CRM programs.
- Balancing short-term revenue against long-term relationship health.
- Measuring customer equity movement.

## Inputs

- Starting customer value.
- Ending customer value.
- Revenue and margin.
- Retention and churn.
- Customer equity changes.

## Repeatable Steps

1. Define the customer asset being measured.
2. Measure value created during the period.
3. Measure value consumed or destroyed.
4. Compare short-term gains with long-term value effects.
5. Feed results into strategy and prioritization.

## Software Artifacts

- Return on customer metric.
- Customer equity dashboard.
- Period-over-period value report.
- CRM program scorecard.
- Value impact review.

## Acceptance Checks

- The system can show value movement, not only activity.
- Short-term campaigns are checked against long-term impact.
- Customer equity is visible at segment or portfolio level.
- Decisions can be reviewed through a customer-value lens.
