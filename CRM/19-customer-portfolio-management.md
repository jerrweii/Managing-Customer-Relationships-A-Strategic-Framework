# Skill 19: Customer Portfolio Management

## Purpose

Manage customers as a portfolio with different strategies for different value and need groups.

## When To Use

- Designing account strategy.
- Allocating sales, service, or success resources.
- Creating differentiated treatment policies.

## Inputs

- Customer value tiers.
- Needs segments.
- Cost-to-serve.
- Strategic fit.
- Growth potential.

## Repeatable Steps

1. Group customers by value, potential, needs, and strategic role.
2. Define the strategy for each portfolio group.
3. Assign service levels, offers, workflows, and ownership.
4. Monitor movement between portfolio groups.
5. Review whether resource allocation matches customer value.

## Software Artifacts

- Customer portfolio view.
- Portfolio segment definitions.
- Account strategy workflow.
- Service-level policy.
- Movement and migration dashboard.

## Acceptance Checks

- Portfolio groups drive different actions.
- Resource allocation is explicit.
- Customers can move between groups as value or needs change.
- Portfolio strategy is reviewed with outcome data.
